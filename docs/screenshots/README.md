# Screenshots

Thư mục này chứa các screenshots của ứng dụng.

## Cách chụp screenshots

Sau khi chạy ứng dụng, chụp các màn hình sau:

1. **homepage.png** - Trang chủ với form tạo short URL
2. **dashboard.png** - Dashboard tổng quan
3. **analytics-charts.png** - Các biểu đồ analytics
4. **link-detail.png** - Trang chi tiết của một link
5. **redirect.png** - Trang chuyển hướng với countdown

## Recommended Tools

- **macOS**: Cmd + Shift + 4
- **Windows**: Snipping Tool hoặc Win + Shift + S
- **Browser Extension**: Full Page Screen Capture

## Kích thước đề xuất

- Width: 1200px - 1400px
- Format: PNG hoặc WebP
- Compress images để giảm size
